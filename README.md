# IdeaVault - Startup Idea Sharing Platform

A modern web platform where innovators share, discover, and validate startup ideas through community interaction and collaboration.

## 🌐 Live Site

## ✨ Features

- 🚀 **Share Startup Ideas** — Submit your startup ideas with detailed problem statements, proposed solutions, target audience, and estimated budgets
- 🔍 **Explore & Search** — Browse all ideas with real-time search by title and filter by category (Tech, Health, AI, Education, Finance, Environment)
- 💬 **Community Comments** — Engage with ideas through a full comment system with add, edit, and delete functionality
- 🔐 **Secure Authentication** — Email/password and Google OAuth login powered by Better Auth with JWT-protected API routes
- 🌙 **Dark / Light Theme** — Toggle between dark and light mode globally with persistent preference
- 📱 **Fully Responsive** — Optimized for mobile, tablet, and desktop screens
- 👤 **Personal Dashboard** — Manage your own ideas (create, update, delete) and track all your interactions
- ⚡ **Fast & Modern UI** — Built with Next.js 16, Tailwind CSS, and HeroUI components

## 🛠️ Tech Stack

- **Framework:** Next.js 16 (App Router)
- **Styling:** Tailwind CSS v4
- **UI Components:** HeroUI v3
- **Authentication:** Better Auth + JWT
- **Icons:** Gravity UI Icons
- **State Management:** React Context API

## 🚀 Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Create `.env.local` with your environment variables
4. Run development server: `npm run dev`
5. Open [http://localhost:3000](http://localhost:3000)
